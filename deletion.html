User Data Deletion Policy - Sheristan

1. User Right to Deletion

Users of Sheristan have the right to delete their account and associated personal data permanently at any time.

2. Data Retention and Deletion Timing

Sheristan stores personal data only for as long as the user maintains an active account. Once the user deletes their account, all associated personal data is permanently removed from our systems.

3. Scope of Data Deleted

Upon successful account deletion, the following data associated with the user account is permanently removed:

  • Name, email address, phone number (if applicable), username, and profile picture
  • User-generated content, including shers/quotes, likes, and comments
  • Device token
  • Any other personal data stored within our systems

4. How to Request Deletion (Procedure)

Users must initiate the deletion request through the Sheristan mobile application. Since Sheristan uses Google and Facebook authentication, deletion requests must be submitted via the same app interface through which the account was created.

  • In-App Deletion: Navigate to the Account Settings section in the Sheristan app and select the option to “Delete your account and data permanently.”

5. Third-Party Data Deletion

Sheristan integrates with Google and Facebook strictly for authentication purposes. Data collected by these third-party services is governed by their own privacy policies.

If you wish to delete data held by Google or Facebook, you must initiate deletion requests directly with those service providers through their respective platforms.
